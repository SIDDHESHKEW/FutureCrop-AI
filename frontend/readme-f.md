# Frontend Run Guide

## Location
Run commands from the frontend folder:
- E:/Desktop E-Drive/Projects/Hackthon projects/FarmerCrop AI/frontend

## Prerequisites
- Node.js installed
- npm available

## Install dependencies

```powershell
npm install
```

## Start frontend dev server

```powershell
npm run dev
```

## Open app
- Vite usually runs on http://127.0.0.1:5173 or http://localhost:5173
- Use the URL printed in terminal after startup

## Build for production

```powershell
npm run build
```

## Preview production build

```powershell
npm run preview
```
